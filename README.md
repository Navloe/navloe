# Backend Navloe

### installation guide

### step 1: install package
```terminal
pnpm i
```

### step 2: generate prisma
```terminal
npx prisma generate
```

### step 3: create database
```terminal
npx prisma db push
```

### step 4: seeding data
```terminal
npx prisma db seed
```

### step 5: run project
```terminal
pnpm dev
```
